慶法輪大法日 韓國首爾舉行盛大遊行

2019 / 5 / 16

二零一九年五月十二日，韓國首爾中心地帶—市政府前的首爾廣場四週彩旗飄揚，悠揚美妙的音樂繚繞在廣場上空。在社團法人韓國法輪大法佛學會的主辦下，上千名韓國部份法輪大法學員匯聚於此，同慶第二十屆「世界法輪大法日」、法輪大法弘傳二十七週年暨創始人李洪志先生華誕。

五月十三日被定為「世界法輪大法日」，這一天也是法輪功創始人李洪志先生首次公開向世人傳播法輪大法的日子。每年的這個時候世界各國的法輪功學員都會舉行各種慶典活動，傳遞「真、善、忍」的美好，同時展現他們二十年如一日和平理性堅持不懈反迫害的精神，更以此向李洪志師父表達感恩。

當天上午九點三十分，法輪功學員在首爾廣場演示五套功法，拉開了慶祝「世界法輪大法日」的序幕。優美祥和的煉功音樂飄盪在廣場上空，舒緩柔和的煉功動作展現了法輪大法的美好與殊勝，吸引了現場民眾諮詢並學煉功法。


圖1～11：二零一九年五月十二日，韓國法輪功學員在首爾廣場集體煉功慶祝世界法輪大法日。


圖12：參加慶典的韓國法輪功學員齊聲恭祝李洪志師父生日快樂，並合影留念。


圖13：天國樂團在集會現場演出。


圖14：腰鼓隊在為觀眾表演。

在上午的紀念集會上由法輪功學員組成的天國樂團和腰鼓隊還為大家獻上精彩的慶祝表演, 來賓也前來賀詞，整個廣場沉浸在一片祥和的氛圍中。下午兩點，參加慶典的法輪功學員集體恭祝李洪志師父生日快樂並合影留念。之後舉行了盛大的遊行。


圖說：集會上，活動主辦方社團法人韓國法輪大法佛學會的權洪大會長在致紀念詞。

集會上，活動主辦方社團法人韓國法輪大法佛學會的權洪大會長致紀念詞，他說：「法輪大法也被稱為法輪功，自一九九二年在中國公開傳開以後，因功效顯著，在短短的7年間就有一億人在修煉，這數字遠遠超過了當時的共產黨人數5600萬。 時任中共黨魁江澤民對此十分妒忌，同時，同化「真善忍」的好人越來越多也引起了他的恐慌，因此他不顧全體政治局常務委員的反對，一意孤行地開始了對法輪功的全面打壓。」

權會長表示：「被中國共產黨欺騙的世人通過過去27年來法輪功學員們和平理性堅持不懈反迫害的歷史知道了法輪大法的真相，希望能夠通過法輪功的修煉恢復人性和道德，從而在這片土地上實現真正的和平」。

「堅守這份勇氣 讓更多的人了解真相」

國際戰略研究所所長金侹玟博士應邀出席了本次「世界法輪大法日」的慶祝活動，他在致詞時，向參與此次慶祝活動的所有人深表敬意。


圖15：國際戰略研究所所長金侹玟博士應邀出席了本次「世界法輪大法日」的慶祝活動並發表演講。

金侹玟曾在二零零零年前往中國留學，當時對法輪功一無所知的他，在那裏第一次聽說了法輪功，但是聽到的卻是對法輪功誣蔑性的說法。「可是後來我往返於中韓兩國之間，越來越覺得奇怪，於是我自己上網搜索了解，之後才知道了法輪功是被中共迫害的，還有諸如（中共的）器官交易等，都是真的。」他說：「針對中共的暴行，在首爾廣場這樣的著名公共場地，法輪功學員這樣勇敢地站出來，對此，我聲援支持這些學員來到這裏。我們國家和其它一些國家，因為懼怕中共政權，擔心貿易受到威脅而閉口不言，我覺得這是非常怯懦的行為。」

「時至今日才知道原來在韓國有這麼多人修煉法輪功。如果各位堅守這份勇氣，將這樣的活動繼續辦下去，我相信總有一天，韓國的一般民眾都會知道中國共產黨的暴行，了解到這場鎮壓。」

喜得大法家庭合樂融融 全家謝師恩

曾擔任韓國鐵道公社設施處處長的鄭寅軾先生帶全家來到慶祝現場，親身感受法輪大法超常威力的他興奮地說：「法輪功即神奇又無比珍貴。修煉前，我的右肩和胳膊又痛又麻，中、西醫都接受過治療，卻不見好轉，在閱讀《轉法輪》並煉功不久後，痛症就消失了。」


圖16：韓國鐵道公社設施處前處長鄭寅軾先生全家及外孫對李洪志先生表達由衷的感恩。

鄭寅軾從二零一四年開始修煉，他進一步說，「修煉不久後的一次打坐過程中，感覺到有一種涼涼的物質從疼痛的那隻胳膊的右邊出去，之後苦不堪言的頑疾就消失無蹤了。那一瞬間太神奇了，到現在都還記憶猶新。」

鄭寅軾的太太金朝伊是韓國慶尚北道容州市文化觀光解說員，她比鄭寅軾早開始修煉法輪功。她在修煉前性格剛強，甚麼都想做得完美，為此常感覺身心疲憊，自從修煉後變得她身心愉快，「修煉使我學會了遇事向內找自己哪裏有問題了，即使身處困境，也能放下自己去想問題、包容別人，心也隨之變得輕鬆愉悅。」在大法中深深受益的金朝伊非常希望自己的一雙兒女也能修煉。女兒鄭賢珍一開始對修煉半信半疑，但是她還是參加了法輪功九天講法學習班，「當時我一邊打瞌睡一邊聽，但很神奇的是，只要出現之前我持有懷疑、不太理解的部份時，總會從瞌睡中清醒過來，聽到法中明確的解答。我的心結也隨之解開了。」從此她走上修煉之路。

鄭賢珍時刻按照法輪功的「真、善、忍」法理來要求自己，遇事找自己哪裏做錯了，暴躁的性格逐漸變得溫柔了，對子女也能循循善誘的教育，原本反對她修煉的丈夫看到這些，不僅支持她修煉，還支持兒女修煉，家庭變得融洽溫暖了。

特別令鄭寅軾夫婦欣慰的是，性格內向、缺乏社交能力的兒子鄭慜揆，修煉後， 找到適合自己的工作，鄭慜揆靦腆地說：「雖然工作辛苦一點，但心不那麼累了，內心很舒暢。」現今，這一家修煉人對李洪志先生心存感激，「師父的洪恩，千言萬語，不知從何說起，如果沒有修煉，也許每天還過著掙扎、苦累的生活，銘記師父的恩惠，不忘初衷，做一個合格的大法弟子。」

難以治癒的肝硬化因修煉而痊癒

李康明在韓國從事三十年警察工作並擔任要職，特殊的工作令他時刻繃緊神經，為了緩解壓力，時常用煙酒來麻痺自己，「那時我長期抽煙喝酒，結果導致了肝硬化。醫生當時對我說，『肝硬化是不可能康復的，只能用藥物阻止它進一步惡化。』」


圖17： 法輪功學員李康明向師父李洪志先生表達感恩之情。

二零零五年的某一天，李康明參觀了首爾麻浦區文化會館舉辦的「真善忍美術展」，自此走入修煉，短短的時間內，他的身體發生了驚奇的變化。「修煉三個月後，我原本千斤重的身體輕鬆得彷彿可以飛起來，我立即停止了治療肝硬化的藥物。」他說，「我立刻去保健所體檢，醫生驚訝地說，你的肝和骨質密度就像三十多歲的人一樣年輕。」

此後，他全身心投入修煉，經常向親友們講述修煉法輪功的神奇。他面帶笑容說，「我家最小的孫女還在上小學，每次看見我時都邊跑邊喊『爺爺，法輪大法好，真善忍好』來和我打招呼。」今年七十八歲的他感嘆道，「修煉後回首往事才醒悟，那些工作與社會中，由於尖銳的爭鬥之心引起的爭執和矛盾是多麼的無意義啊。」

今天，他真誠地向法輪功創始人表達自己的感恩：「李洪志師父賦予了我全新的生命和健康的身心，以及幸福的家庭，我真心感謝師父！」

少年監獄教導員: 感恩師父引導我走向光明的未來

目前在韓國一所少年監獄當教導員的金映杓，二零零七年開始修煉，可他曾是一名酒精中毒患者，一米七左右的個子，體重還不到五十公斤。「那時的我沒有酒根本就撐不過三天，一喝就喝到爛醉為止，根本直不起身來。身子就這樣一點點被毀了。」後來，在姪子的勸導下，他開始閱讀《轉法輪》並走入修煉：「自從每天看《轉法輪》後，根本就想不起來喝酒的事，自然也就擺脫了酒精中毒的狀態，真的很神奇！」


圖18： 法輪功學員金映杓在世界法輪大法日表達對師父的感謝。

談到修煉後的變化，他喜悅地表示，「我原來的皮膚被酒精熏得發黑，修煉後變得白裏透紅，體重也恢復了正常水平。原本刻薄的性格也變得大度，胸懷變得寬廣，明確了人生的方向，做個好人才是最重要的。現在每天都過得快樂而充實。」金映杓顯著的身心變化也影響了身邊的人：「父母和親戚看到我這樣明顯的變化，都很支持我修煉，我在工作中也秉持修煉人的正念，與同事和睦相處，身邊的同事也都很支持大法。」

談及今天參加活動的感受，他說：「我很高興能夠在世界法輪大法日，向市民們展示大法的美好。在這裏向師父奉上誠摯的敬意，感謝師父引導我走向光明的未來，恭祝師父生日快樂。」

遊行隊伍壯觀宏大 吸引眾多行人駐足觀看

下午兩點三十分，遊行活動拉開序幕。遊行路線，從首爾廣場出發途徑市議會、清溪川廣場、光化門、仁寺洞、鐘閣、明洞及樂天百貨店，全長4.5公里。

身穿華麗韓服的三名女學員打出「慶祝法輪大法洪傳27週年」的橫幅，緊接著是在遊行隊伍中打頭陣的天國樂團；「法輪大法」大型橫幅、花車、煉功演示隊伍、腰鼓隊跟隨其後，參加遊行行列的法輪功學員們各手舉橫幅向路人展示大法的美好、要求停止迫害，整個遊行隊伍壯觀而宏大，吸引了無數行人駐足觀看和讚歎。


圖19～31：慶祝法輪大法弘傳二十七週年，韓國法輪功學員在首爾市中心舉行盛大遊行活動。

路邊的行人紛紛拿出手機或拍照或錄像，幾位來自中國遊客邊錄像邊說，「哇！隊伍這麼長、這麼壯觀啊。」其中一位女士對同伴表示，法輪功世界各地都有，曾在加拿大也看過，有信仰一定有它的好處，信仰是誰都不能虐殺的。

來自中國的劉先生（化名），一直靜靜地在道邊的欄杆外觀看著遊行隊伍，此情此景，令他想起對中共為甚麼以各種手段封殺人們了解真相的渠道，就是「做賊心虛」。他贊同中國民眾的普遍想法，中共媒體不是為人民服務，而是為「黨的需要服務」。

前來參觀景福宮的韓國首爾新木中學學生李宰沅看到法輪功學員的遊行隊伍後說：「整個遊行很壯觀，也很華麗。」他的朋友們笑容燦爛地表示，「遊行隊伍讓人有一種非常溫暖的感覺。」


圖32：韓國首爾新木中學學生們說，法輪功的遊行隊伍給人有一種非常溫暖的感覺。

遊行隊伍進入仁寺洞步行街時，和母親在此散步的崔俸碩看到在這個遊行群體裏，上到年老者，下到年輕人，各個年齡層的人都有，「留下了深刻的印象」。對在中國，法輪功學員僅因修煉就遭受群體性的迫害，他「感到非常痛心」。


圖33：週末外出遊玩時碰到遊行隊伍的崔俸碩（左）和母親。

當他和母親聽說今天遊行隊伍大部份是韓國人時，非常驚訝，表示以後也想修煉法輪功。
